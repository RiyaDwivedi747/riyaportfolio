### 📝 Project Overview
A premium, high-performance portfolio website built for **Riya Dwivedi**, a detail-oriented Data Analyst. This project showcases a modern UI/UX with smooth animations, responsive layouts, and a professional dark luxury aesthetic.

> [!TIP]
> **View the live site here:** [riyaportfolio.live](https://RiyaDwivedi747.github.io/riyaportfolio)

---

### ✨ Key Features
- 💎 **Premium UI**: Dark luxury theme with glassmorphism and neon accents.
- 🚀 **Performance**: Blazing fast load times powered by Vite.
- 🎨 **Animations**: Smooth, interactive entrance and scroll animations using Framer Motion.
- 📱 **Fully Responsive**: Optimized for every screen size from mobile to desktop.
- 📧 **Contact Integration**: Working contact form with EmailJS integration.
- 📊 **Dynamic Sections**: Hero, About, Skills (Data & Viz), Projects, and Experience.

### 🚀 Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/RiyaDwivedi747/riyaportfolio.git
   ```

2. **Navigate to the directory**
   ```bash
   cd riyaportfolio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```
